[ 핵심 명령어 ]

1. VSCODE에서 터미널 열기 : ctrl+` (보기>터미널)
2. 버전확인 :
    1) node -v : 노드JS 버전확인
    2) npm -v : npm 버전확인
    3) vue --version :  뷰CLI 버전
    -> 미설치시 명령어를 인식하지 못함!
3. 기본 명령어 내용 구분하기
    (1) 명령어
        1) install -> i : 설치명령어
        2) uninstall -> uni : 삭제명령어
    (2) 옵션
        1) -g : 전역설치옵션(안쓰면 현재폴더에 설치됨)
            -> 전역은 어디? nodejs 프로그램 설치 폴더
        2) --f : --force 강제설치 옵션
