[ 핵심 명령어 ]

1. 터미널 조작
    1) VSCODE에서 터미널 열기 : ctrl+` (보기>터미널)
    2) 터미널 창에서 위아래 방향키는 조금전 명령어 다시부르기
    3) 언제든지 esc버튼을 누르면 빠져나감(명령어지워짐)
    4) 개발서버 구동중인 경우 닫고 싶을때는?
        ctrl+C 단축키사용하면 닫겠니? 라고 물어본다! y 엔터!
        -> C는 Close임!

2. 버전확인 :
    1) node -v : 노드JS 버전확인
    2) npm -v : npm 버전확인
    3) vue --version :  뷰CLI 버전
    -> 미설치시 명령어를 인식하지 못함!

3. 기본 명령어 내용 구분하기
    (1) 명령어
        1) install -> i : 설치명령어
        2) uninstall -> uni : 삭제명령어
        3) cd (change directory) : 폴더변경 명령어
        -> cd ../ 한단계위, cd 전체경로 -> 이동
        -> cd (여기에 드래그엔 드롭)

    (2) 옵션 : 팩키지 앞/뒤 어디나 사용가능!
        마이너스 하나(-) 기본옵션
        마이너스 둘(--) 추가옵션
        1) -g : 전역설치옵션(안쓰면 현재폴더에 설치됨)
            -> 전역은 어디? nodejs 프로그램 설치 폴더
        2) --f : --force 강제설치 옵션
        3) --save : 의존성에 저장하는 옵션(기본값)
            -> 의존성은 package.json의 dependencies 항목에
            설치된 파일을 등록한다는 의미임!

            -> 참고로 의존성은 개발서버용 의존성이 있음!
            devDependencies 항목임. 개발환경에서만 쓰는 확장앱
            ->>> 개발서버용 의존성에 등록하고 싶을때는
            반드시 의존성 저장옵션을 아래와 같이 사용한다!
            --save-dev
