Flex 상단영역 3가지 배치 예시

1. 첫박스 200px, 중간박스 자동, 끝박스 150px

첫박스
200px
중간박스
자동
끝박스
150px
설명: 첫 박스와 끝 박스는 고정 너비, 중간 박스가 남은 공간을 차지합니다.

2. 첫박스와 끝박스 균등분할, 중간박스 200px (로고 중앙)

첫박스
1fr
중간박스
200px
(로고)
끝박스
1fr
설명: 양쪽 박스가 공간을 균등하게 차지하고, 중간 박스(로고)가 정중앙에 위치합니다.

3. 첫박스 200px, 중간박스 300px, 끝박스 150px (중간박스 중앙배치)

첫박스
200px
중간박스
300px
(로고)
끝박스
150px
설명: 모든 박스가 고정 너비이며, justify-content: space-between으로 중간박스가 중앙에 배치됩니다.
(중간박스가 정확히 중앙에 위치하지는 않음)

3. 첫박스와 끝박스 등분할
(첫박스 자식박스 200px,끝박스 자식박스 150px),
중간박스 300px (중간박스 정중앙배치)

첫박스
200px
중간박스
200px(로고)
끝박스
150px
설명:
좌/우를 각각 flex:1로 등분할 좌: 내부에 200px 박스(왼쪽 정렬) 가운데: 고정폭 로고, 우: 내부에 150px 박스(오른쪽 정렬)
로고가 항상 "정중앙"에 위치!
